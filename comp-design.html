Simulated Polaroid — Process Notes

Tools

  • Python
    • Pillow (PIL) (layout construction, texture layering, compositing)
    • random (stochastic variation)
    • argparse (parameterized output)
  • Photoshop
  • Midjourney

I created a synthetic photograph of an Ophrys apifera (Bee Orchid) in bloom using Midjourney. Then, I created a worn Polaroid frame in Photoshop by layering texture overlays. Next, I wrote a Python script (download) to treat the image as a material object subject to wear, exposure, and chemical instability.

Simulated Daguerreotype — Process Notes

Tools

  • Python
    • NumPy (pixel-level tonal remapping, metallic color mapping, vignette math)
    • Pillow (PIL) (grayscale conversion, compositing, filters, alpha masks)
    • math (directional sheen and radial calculations)
  • Photoshop
  • Midjourney

I used Midjourney to create a synthetic photograph of an Helenium autumnale (Sneezeweed) in bloom. I then processed the image using a Python script that converts the photograph to grayscale, reshapes its tonal curve, and maps it onto a simulated silver plate surface with sheen, vignette falloff, and tarnish artifacts.

Simulated Heat Map Self Portrait — Process Notes

Tools

  • Python
    • NumPy (depth-to-color mapping, numeric normalization)
    • Pillow (PIL) (image I/O and format conversion)
  • Photoshop

I began with a photographic self-portrait and constructed a depth map in Photoshop by segmenting the image into spatial regions. I then processed this depth map with a Python script that reads pixel-level depth values and remaps them to a false-color heat visualization.
